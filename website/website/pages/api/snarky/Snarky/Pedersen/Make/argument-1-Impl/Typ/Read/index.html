Module Typ.Read

A 'value Read.t describes reading values back out of the R1CS variables, so that the prover can use them in As_prover blocks. It is a monad, which lets us combine these values to create more complex ones.

For example, we can create a record containing the results of some reads by writing

type ('a, 'b) t = {a : 'a; b : 'b}

let read_t (read_a : 'a Read.t) (read_b : 'b Read.t) : ('a, 'b) t =
  let open Read in
  let%map a = read_a
  and b = read_b
  in
  {a; b}
